                    UDIT SINGH PARIHAR Seidu Abdulai • a year ago • edited

                    It is basically due to:
                    1. The XOR of a number with itself is zero.
                    A ^ A = 0
                    2. The XOR of a number with zero is the number itself.
                    A ^ 0 = A
                    3. The order in which you do XOR is does not matter.
                    A ^ B = B ^ A
                    So, all the even occurences number would give 0 by combining in pair. And finally when you combine 0 with the odd occured number, you would get 0.
